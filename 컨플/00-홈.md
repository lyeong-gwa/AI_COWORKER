# AI 업무도우미

**AI 기반 업무 관리 및 워크플로우 자동화 도구**

---

## 소개

AI 업무도우미는 업무 관리, 지식 관리, 워크플로우 자동화를 하나의 플랫폼에서 제공하는 통합 도구입니다. AI 채팅 어시스턴트가 지식 베이스를 기반으로 업무를 자동 생성하고, 관련 문서를 참조하여 답변을 작성하며, 워크플로우를 통해 반복 업무를 자동화합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **태스크 보드** | Trello 스타일 칸반 보드로 업무를 시각적으로 관리 |
| **지식 베이스** | 마크다운 문서를 벡터 DB에 동기화하여 AI 검색 지원 |
| **도구 관리** | API 호출, 파일 처리, 코드 실행 등 재사용 가능한 도구 라이브러리 |
| **노드 관리** | AI 노드를 정의하고 도구를 바인딩하여 자동화 단위 구성 |
| **워크플로우** | AI 노드를 연결하여 비주얼 워크플로우를 구성하고 실행 |
| **AI 어시스턴트** | 모든 페이지에서 사용 가능한 지식 기반 AI 챗봇 |

---

## 기술 스택

| 구분 | 기술 |
|------|------|
| **Frontend** | React 19 + TypeScript + Vite 7 + Tailwind CSS (다크 테마) + @xyflow/react |
| **Backend** | FastAPI + SQLAlchemy (비동기, SQLite) + ChromaDB (벡터 DB) + ONNX 임베딩 |
| **AI** | LLM API (GPT-4o) + RAG (Retrieval-Augmented Generation) |

---

## 공통 UX 기능

- **사이드바 네비게이션**: 5개 페이지 간 빠른 이동, Ctrl+1~5 단축키 지원
- **API 연결 상태 표시**: 사이드바에서 백엔드 연결 상태를 실시간 확인
- **반응형 레이아웃**: 데스크톱 및 모바일 환경 모두 지원
- **토스트 알림**: 작업 성공/실패/경고/정보 메시지 표시
- **마크다운 렌더링**: 테이블, 목록, 볼드, 코드블록 등 지원
- **폼 유효성 검사**: 입력 필드 문자 수 카운터 및 제한
- **키보드 단축키**: Escape로 모달 닫기, Ctrl+N으로 새 항목 생성
- **404 에러 페이지**: 잘못된 URL 접근 시 안내 페이지 표시

## 페이지 목록

### 1. [태스크 보드](02-태스크-보드.md)
프로젝트 태스크를 칸반 보드 형태로 관리합니다. 5단계 상태 관리, 드래그앤드롭, 검색/필터, 우선순위, 태그, TODO 체크리스트, 댓글, 활동 이력을 지원합니다.

- [태스크 AI 기능](02-1-태스크-AI기능.md) - AI 자동 태스크 생성, 지식 기반 추론, 참조 자료

### 2. [지식 베이스](03-지식-베이스.md)
마크다운 문서를 벡터 DB에 동기화하여 AI가 검색하고 활용할 수 있는 지식 저장소를 관리합니다.

### 3. [도구 관리](04-도구-관리.md)
API 호출기, 파일 읽기/쓰기, 코드 실행, DB 쿼리 등 재사용 가능한 도구를 등록하고 관리합니다.

### 4. [노드 관리](05-노드-관리.md)
AI 노드를 정의하고, 도구를 바인딩하며, LLM 모델과 입출력 스키마를 설정합니다.

### 5. [워크플로우](06-워크플로우.md)
AI 노드를 시각적 캔버스에서 연결하여 자동화 워크플로우를 구성하고 실행합니다.

### 6. [AI 어시스턴트](07-AI-어시스턴트.md)
모든 페이지에서 사용 가능한 AI 챗봇으로, 지식 베이스를 기반으로 질문에 답변하고 업무를 자동 생성합니다.

### 7. [사용자 시나리오](08-사용자-시나리오.md)
실제 사용 흐름을 시나리오별로 정리합니다. 사용자 입력 → AI 처리 → 결과 확인까지의 전체 과정을 스크린샷과 함께 안내합니다.
